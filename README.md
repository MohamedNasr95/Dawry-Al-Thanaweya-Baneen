# Dawry-Al-Thanaweya-Baneen

## Game Rules:
    1. Players have 2 matches per week.
    2. Matches will be played with best of 3 system.
    3. Round consists of 10 points.
    4. Players have to choose and accept 2 Judges for the match.
    5. Both players have to accept judges decisions. 
    6. Judges must focus on stepping on the line.
    7. Every player plays 2 matches with every opponent
    
## Points distribution:
    1. Winning a match 2-0 >> 2 points.
    2. Winning a match 2-1 >> 1 point.
    3. Losing a match      >> 0 points.
